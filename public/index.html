OAMK Notes API

This is the API powering OAMK Notes, a first year university project for the Oulu University of Applied Sciences, created by Tavis Sinclair, Aleksandar Raynov, Kai Oraviita and Bars Margetsch.

If you would like to know more, click here for the GitHub repository, or here to go straight to the documentation.

This API and the associated MySQL database is hosted on Microsoft Azure.
